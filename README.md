# Job Vacancy System - User Manual

This documentation provides a comprehensive guide on how to navigate and utilize the Job Vacancy System. The platform is divided into three distinct user perspectives: **Admin**, **Person In Charge (PIC)**, and **Applicant**.

---

## Authentication and System Access

To access the system dashboard, all users must authenticate via the unified Login Interface. 

### Login Procedure
1. Navigate to the system login page.
2. Select the appropriate user type (**Admin**, **PIC**, or **Applicant**) utilizing the provided radio buttons.
3. Input the registered **Username** and **Password**.
4. Click the **Submit** button to route to the respective dashboard.

### Registration Protocols
* **Admin:** Pre-configured within the system database; no self-registration option is available.
* **Applicants:** First-time users must complete the registration form via the "Register" link prior to their initial login.
* **PIC (Companies):** Registration must be initiated by an Admin. The Admin submits the organization's details, after which the PIC can log in using the designated credentials.

---

## 1. Admin Portal

The Admin portal provides oversight of the system's applications and manages organizational credentials.

### 1.1 Application Management & Analytics
* **Dashboard Overview:** View a consolidated summary of all incoming job applications from applicants.
* **Monthly Reporting:** Generate monthly analytical reports. The system automatically processes the data to produce:
  * A structured bar chart visualizing application trends.
  * A ranked list mapping job applications from the highest to lowest volume, categorized by specific company demands.

### 1.2 PIC Management
* **Registering a PIC:** 
  1. Navigate to the **Register PIC** section.
  2. Populate the registration form with the mandatory details: *Company Name, Company Email, Company Phone Number, Username, Password, Company Address, State, and City*.
  3. Click the **Submit** button to save the records securely into the database.
* **Modifying Accounts:** Access the **PIC Details Section** to update existing company profiles or permanently remove PIC records from the system.

### 1.3 System Exit
* Click the **Log Out** option to securely terminate the active administrative session.

---

## 2. Applicant Portal

The Applicant portal allows job seekers to discover vacancies, manage applications, and maintain personal profiles.

### 2.1 Job Search & Application
* **Job Discovery:** Navigate to the **Job Vacancy Section** to search for openings. Queries can be filtered by *Location, Job Title,* or *Salary Range*.
* **Submitting an Application:** Select a vacant position and attach a valid resume file to complete the submission.

### 2.2 Application Tracking
* Navigate to the **Application Status** section to monitor current submissions.
* **Pending Status:** If an application remains in a pending state, the applicant retains the option to delete the submission.
* **Approved or Rejected Status:** Once a decision is finalized, the record becomes permanent and cannot be deleted. 
* **Interview Notifications:** Approved applications will display the following system message: *"We will reach you soon via email/whatapps for an interview meeting"*.

### 2.3 Profile Management
1. Navigate to the **Update Profile** section.
2. Modify personal information as required.
3. Click the **Update** button.
4. Upon successful data commitment, a confirmation pop-up will state: *"PROFILE UPDATED SUCCESSFULLY !!"*.

### 2.4 System Exit
* Click the **Log Out** button to securely close the session.

---

## 3. Person In Charge (PIC) Portal

The PIC portal enables company representatives to publish employment opportunities and evaluate incoming talent.

### 3.1 Job Posting Management
* **Creating a Posting:** Complete the job creation form with the following criteria: *Position Image, Job Position, Company Name, Location, Language Preferences, Educational Level, Years of Working Experience, Working Days, Salary Range,* and *Job Description*.
* **Post History:** Published listings are automatically indexed and displayed chronologically within the main PIC Dashboard for archival tracking.

### 3.2 Candidate Evaluation
* Navigate to the **Applicant** management section.
* Review candidate submissions and utilize the interface to update the application status to either **Approved** or **Rejected**.

### 3.3 System Exit
* Click the **Log Out** button to terminate the session and clear credentials from the active browser state.
